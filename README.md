# Walmart---Sales
Peguei um dataset do Walmart com 6435 registros de 45 lojas entre 2010 e 2012. Tratei os dados do Excel, criei um dashboard no Power BI e estruturei um Banco de Dados SQL com Python para automatizar a análise. As principais descobertas foram que a loja 4 foi a maior vendedora e que semanas de feriado vendem em média 4.5% a mais
